Pull request
1. git checkout -b < feature_branch >
2. git push origin < branch >
3. create a pull request on Github
4. Review code
    4.1: Review code online (Github)
    4.2: Fetch branch into local to test offline (Opitional)
    4.3: Approve the pull request

5. Merge to master
